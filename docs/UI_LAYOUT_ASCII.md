## 响应式设计说明

所有页面都支持响应式布局：

- **桌面端** (>1024px): 完整布局，侧边栏展开
- **平板端** (768px-1024px): 紧凑布局，侧边栏可折叠
- **移动端** (<768px): 单列布局，堆叠显示
